Задание 1
-----

Заничительно проще создать репозиторий на саййте, потом выполнить следуюее:  
 1. клонировать на локальный ПК используюя git clone
 2. создать файлик readme.md
 3. git add .
 4. git commit -m "add readme.md"
 5. git push
